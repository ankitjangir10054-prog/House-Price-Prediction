# 🏠 AI House Price Prediction

This project predicts house prices using a CatBoost Machine Learning model.

## Features

- AI-powered house price prediction
- Beautiful responsive UI
- Flask backend
- CatBoost Regressor
- RobustScaler preprocessing

## Tech Stack

- Python
- Flask
- CatBoost
- HTML
- CSS
- JavaScript

## Run Locally

pip install -r requirements.txt

python app.py
